# PwrAgnt

Thread-centric coding agent desktop app.

## Getting Started

1. `pnpm install`
2. `pnpm dev`

## Workspace

- `apps/desktop` - Electron app shell
- `packages/shared` - shared contracts and types
- `packages/agent-core` - agent runtime and domain services

## Testing

- `pnpm test`
- `pnpm typecheck`
